AUTO_RENAME: USER NOTES

- On the BIG app, I would reccomend using MENLO Regular as your font of choice for most accurate ID'ing
- Other font reccomendations listed below

- Try stick to all caps and underscores. It will still work but illegal characters won't reflect in your new folder names
- Avoid massive glare/reflections on phone screen, face towards light sources when capturing slate.
- This tool assumes each folder has been made correctly for a single slate and/or item and will rename accordingly
  it won't be able to discern out mis-foldered or double-foldered items.
- GPU acceleration is enabled by default, user can disable this if not computationable available. Will by-pass if not available
- Centre the slate and avoid full-screening the entire phone in your image.
